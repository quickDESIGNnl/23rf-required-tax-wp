# Required Taxonomies

This repository contains the complete code for the **Required Taxonomies** WordPress plugin. Clone or download the repository as a ZIP to install it on your site without any additional build step.

## Development

* Plugin entry point: [`required-taxonomies.php`](required-taxonomies.php)
* WordPress.org style documentation: [`readme.txt`](readme.txt)

Feel free to open an issue or submit a pull request if you have feature requests.
