# Required Taxonomies

This repository contains the complete code for the **Required Taxonomies** WordPress plugin. Clone or download the repository as a ZIP to install it on your site without any additional build step.

## Features
* Admin matrix under **Settings → Tax verplicht** to pick required taxonomies per post type.
* Client-side publish/save locking for both Gutenberg and Classic editors until all required taxonomies contain at least one term.
* Automatic refresh after saving settings so the matrix immediately reflects your changes.
